---

## 1. 서비스 개요

### 1-1. 핵심 가치 제안

AI 기반 MBTI 검사와 실시간 1:1 매칭 채팅을 통해 새로운 인연을 만들어주는 서비스

**한 문장 정의**
"AI가 분석한 MBTI로 나와 맞는 사람을 찾고, 바로 대화하는 서비스"

### 1-2. 타겟 사용자

- **주 타겟**: 20-30대 MBTI에 관심 있는 사람
- **페르소나**:
    - 기존 MBTI 검사에 지쳐 새로운 방식을 원하는 사람
    - MBTI 기반으로 맞는 사람을 만나고 싶은 사람
    - 가볍게 새로운 인연을 만들고 싶은 사람

### 1-3. 핵심 차별화

1. **순차 적응형 MBTI 검사** - 사람 질문 12개 → AI가 결과 기반 적응형 질문 12개 생성
2. **한국 정서에 맞는 신박한 질문** - 기존 번역체 X, "우울해서 빵샀어 그러면 뭐라 대답할래?" 같은 질문
3. **24개로 끝** - 피로도 최소화, 정확도 최대화
4. **돌발 질문** - 앱 사용 중 간헐적 질문으로 MBTI 보정
5. **실시간 1:1 채팅** - 매칭 즉시 대화 시작
6. **MBTI 기반 매칭** - 궁합/유사 MBTI 알고리즘 적용

## 2. OKR (4주)

### Objective
사용자가 실제로 돈을 지불할 만큼 매력적인 AI 기반 MBTI 매칭 서비스를 검증한다

### Key Results
1. 가입자 100명 달성
2. 추천/레퍼럴로 사용자 20명 이상 유입
3. 주간 활성 사용자 20명 이상 (리텐션 20% 이상)
4. 핵심 가치 검증: 매칭 성사 후 대화 지속률 50% 이상
5. 수익 가능성 검증: 결제 유저 1명 이상 확보

## 3. 실행 계획 (4주)

### 팀 구성 (5명)

| 역할 | 인원 | 담당 |
|------|------|------|
| **Team MBTI** | 2명 | MBTI 검사 |
| ↳ Person A | 1명 | 사람이 만든 질문 (일반 + 돌발) |
| ↳ Person B | 1명 | AI 프롬프트 질문 (일반 + 돌발) |
| **Team Match** | 2명 | 매칭 + 채팅 |
| ↳ Person C | 1명 | matching/ (대기열, 매칭) |
| ↳ Person D | 1명 | chat/ (채팅방, WebSocket) |
| **조장** | 1명 | 전체 서포트 |
| ↳ Person E | 1명 | payment/, referral/ + 병목 해결 |

### Week 1-2: MVP 출시 (5명 병렬)
**목표**: 채팅형 MBTI 검사로 관심 유도 + 매칭/채팅으로 가치 검증

- **Person A**: 사람이 만든 질문 (일반 + 돌발)
- **Person B**: AI 프롬프트 질문 (일반 + 돌발)
- **Person C**: 매칭 로직 (대기열, 랜덤 매칭)
- **Person D**: 채팅 (채팅방, WebSocket, 메시지)
- **조장**: 결제 도메인 + 병목 지원

### Week 3: 매칭 고도화
**목표**: MBTI 기반 매칭 알고리즘으로 매칭 품질 개선

- **Person A**: MBTI 궁합 매칭 알고리즘
- **Person B**: MBTI 유사 매칭 알고리즘
- **Person C,D**: 통합/부하 테스트
- **조장**: 결제 API 연동

### Week 4: 그로스 해킹
**목표**: 서비스가 스스로 확산되고, 누군가는 실제로 돈을 지불하는지 검증

- **Person A,B**: 레퍼럴 기능
- **Person C,D**: 버그 픽스 & UX 개선
- **조장**: 결제 테스트 + 전체 조율

### 보류 (100명 달성 후)
- 커뮤니티/모임 기능 → 유저 없을 때 의미 없음
- AI 개인화 추천 (관심사, 활동 패턴) → 초반 데이터 부족
- 프리미엄 구독 (무제한 매칭 등)

## 4. 핵심 기능

### 4-1. 실시간 1:1 매칭 채팅 (NEW - 핵심)

#### 기능 개요
MBTI 기반으로 매칭된 상대와 실시간 1:1 채팅

#### 매칭 방식
1. **랜덤 매칭** (기본) - MBTI 무관하게 매칭, 상대 MBTI 표시
2. **MBTI 매칭** (Week 3+) - 유사/궁합 MBTI 기반 매칭

#### 무료 vs 유료
- **무료**: 하루 3회 랜덤 매칭
- **유료**: 추가 매칭권 구매 (소액 1회성)

### 4-2. MBTI 검사 (Week 1-2 - 핵심!)

#### 기능 개요
순차 적응형 MBTI 검사 - 총 24개 질문으로 정확도 높은 MBTI 도출

#### 검사 흐름

```
[1단계] 사람이 만든 질문 12개 (Person A)
    │
    │  E/I, S/N, T/F, J/P 각 3개씩
    │  룰베이스 가중치로 MBTI 비중 계산
    │  예: E 60% / I 40%, S 30% / N 70%...
    ↓
[2단계] AI가 만든 질문 12개 (Person B)
    │
    │  1단계 MBTI 비중을 받아서 적응형 질문 생성
    │  애매한 차원(50:50에 가까운)을 집중 검증
    ↓
[결과] 최종 MBTI 도출 + 각 차원별 비중 표시
```

#### 질문 스타일 (차별화)
- **한국 정서에 맞는 신박한 질문** (기존 번역체 X)
- 예시: "우울해서 빵샀어 그러면 뭐라 대답할래?"
- 예시: "친구가 갑자기 울면서 전화왔어. 뭐라고 할래?"

#### 돌발 질문 (앱 사용 중 간헐적 등장)

| 구분 | Person A 담당 | Person B 담당 |
|------|--------------|---------------|
| **돌발 질문** | 사람이 만든 돌발 질문 | AI 프롬프트 돌발 질문 |

- 앱 사용 중 예상치 못한 질문 등장
- 응답 분석 후 MBTI 보정 (예: ESTJ → ISTJ)

#### 특징
- **순차 적응형**: 사람 질문 결과가 AI 질문 생성에 영향
- **24개로 끝**: 피로도 최소화, 정확도 최대화
- **한국형 질문**: 번역체 아닌 한국 정서에 맞는 질문
- **돌발 질문**: 간헐적 추가 질문으로 MBTI 보정
- **결과 저장**: MBTI 도출 후 User.mbti에 저장
- **재검사**: 언제든 재검사 가능

### 4-3. 레퍼럴 시스템 (Week 4)

#### 기능 개요
친구 초대 시 보상 제공

#### 보상
- 초대한 사람: 추가 매칭권 1회
- 초대받은 사람: 추가 매칭권 1회

## 5. 기존 기능 (유지)

### 5-1. MBTI 상담소 (요약)
- AI와 대화형 관계 상담
- 무료 5턴, 프리미엄 10-20턴
- 4개 섹션 분석 결과 (상황 분석, 유형별 특성, 해결 방안, 주의사항)

### 5-2. MBTI 대화 변환기 (요약)
- 메시지를 상대방 MBTI에 맞게 변환
- 3가지 톤: 공손/캐주얼/간결
- 복사해서 바로 사용 가능

## 8. 리스크 & 대응

### 8-1. 제품 리스크

**리스크 1: 매칭 후 대화 안 함**
- 대응: 매칭 시 아이스브레이커 제공
- 대응: MBTI 기반 대화 주제 추천

**리스크 2: 유저 풀 부족으로 매칭 안 됨**
- 대응: 초기에는 대기열 + 알림 시스템
- 대응: 레퍼럴로 빠른 유저 확보

**리스크 3: MBTI 검사 신뢰도 낮음**
- 대응: 사람 질문 + AI 질문 혼합으로 다각도 분석
- 대응: 돌발 질문으로 정확도 향상
- 대응: 재검사 기능으로 사용자가 직접 검증

### 8-2. 비즈니스 리스크

**리스크 4: 결제 전환 안 됨**
- 대응: 무료 3회로 가치 체험 후 자연스럽게 유도
- 대응: 소액 1회성으로 부담 최소화
